API Configuration Overview

Film Translator Generator mendukung beberapa API untuk layanan terjemahan. Dokumen ini menjelaskan cara mendapatkan dan mengkonfigurasi API key untuk setiap layanan.

Catatan Keamanan: API key disimpan dengan enkripsi pada aplikasi ini, tetapi tetap penting untuk menjaga keamanan dan kerahasiaan API key Anda. Jangan bagikan API key dengan orang lain.

Aplikasi menyediakan fitur validasi API key untuk Gemini, OpenAI, Anthropic, dan DeepSeek.

Google Gemini API

Gemini AI adalah model bahasa generatif dari Google yang kuat untuk terjemahan.

Mendapatkan Google Gemini API Key

  1. Kunjungi Google AI Studio
  2. Login dengan akun Google Anda
  3. Klik "Get API key" atau buka bagian API keys pada dashboard
  4. Buat API key baru atau gunakan yang sudah ada
  5. Copy API key untuk digunakan di Film Translator Generator

Model yang Tersedia

  • gemini-2.5-flash-preview-04-17: Model cepat dengan kemampuan terjemahan yang baik
  • gemini-2.5-pro-exp-03-25: Model dengan kualitas terjemahan lebih tinggi (lebih lambat)

Parameter Gemini

Parameter Nilai Default Deskripsi
Temperature 0.2 Mengontrol kreativitas output (nilai lebih rendah = output lebih konsisten)
Top P 0.95 Kontrol nucleaus sampling
Top K 40 Jumlah token top yang dipertimbangkan

OpenAI API

OpenAI adalah penyedia model bahasa seperti GPT-4 dan GPT-3.5 yang memberikan hasil terjemahan berkualitas tinggi.

Mendapatkan OpenAI API Key

  1. Kunjungi OpenAI Platform
  2. Buat akun atau login ke akun yang sudah ada
  3. Buka bagian "API Keys" di dashboard
  4. Klik "Create new secret key"
  5. Beri nama pada key (opsional) dan copy key yang dihasilkan

Model yang Tersedia

  • gpt-4.1: Model flagship terkini dengan kemampuan terjemahan terbaik
  • gpt-4.1-mini: Versi lebih ringan dari GPT-4.1
  • gpt-4.1-nano: Versi paling ringan dari GPT-4.1
  • gpt-4o: Model multimodal dengan kemampuan terjemahan yang sangat baik
  • gpt-4o-mini: Versi lebih ringan dari GPT-4o
  • gpt-4-turbo-2024-04-09: Versi turbo dari generasi sebelumnya
  • gpt-3.5-turbo: Model ringan dengan biaya terendah

Anthropic API

Anthropic Claude adalah model AI yang kuat dengan kemampuan terjemahan yang sangat baik.

Mendapatkan Anthropic API Key

  1. Kunjungi Anthropic Console
  2. Buat akun atau login ke akun yang sudah ada
  3. Buka bagian "API Keys"
  4. Klik "Create Key" dan ikuti petunjuk untuk membuat API key baru
  5. Copy API key yang dihasilkan

Model yang Tersedia

  • claude-3-opus-20240229: Model Claude 3 Opus (kualitas tertinggi)
  • claude-3-5-sonnet-20240620: Claude 3.5 Sonnet
  • claude-3-7-sonnet-20250219: Claude 3.7 Sonnet
  • claude-3-5-sonnet-20241022: Claude 3.5 Sonnet (versi alternatif)
  • claude-3-5-haiku-20241022: Claude 3.5 Haiku (model ringan)

DeepSeek API

DeepSeek menawarkan model bahasa alternatif yang mungkin berguna untuk terjemahan bahasa tertentu.

Mendapatkan DeepSeek API Key

  1. Kunjungi DeepSeek Platform
  2. Buat akun atau login ke akun yang sudah ada
  3. Buka bagian API atau developer settings
  4. Ikuti petunjuk untuk membuat API key baru
  5. Copy API key yang dihasilkan

Model yang Tersedia

  • deepseek-chat: Model dasar DeepSeek untuk terjemahan

Keamanan API Key

Film Translator Generator mengimplementasikan enkripsi untuk semua API key yang disimpan dalam file konfigurasi dan proyek.

Fitur Keamanan

  • Enkripsi Lokal: API key dienkripsi menggunakan ID mesin sebagai seed enkripsi
  • Password Opsional: Anda dapat menambahkan password tambahan untuk meningkatkan keamanan
  • Tidak Berbagi Key: API key tidak pernah dikirimkan ke server pihak ketiga

Praktik Keamanan Terbaik

  • Gunakan password tambahan untuk enkripsi jika proyek akan dibagikan
  • Jangan menggunakan API key yang sama untuk aplikasi lain
  • Monitoring penggunaan API key secara berkala
  • Rotasi API key secara berkala

Manajemen Biaya

Penggunaan API terjemahan akan dikenakan biaya tergantung pada penyedia layanan. Berikut adalah beberapa strategi untuk mengelola biaya:

Tips Mengelola Biaya

  • Gunakan model yang lebih ringan untuk video panjang
  • Atur batasan penggunaan pada dashboard penyedia layanan
  • Untuk pengujian, gunakan file video yang lebih pendek
  • Gunakan OpenAI GPT-3.5-Turbo atau Gemini Flash untuk biaya lebih rendah

Film Translator Generator tidak mengumpulkan atau mengakses informasi penagihan Anda. Semua biaya API dikelola secara langsung melalui penyedia layanan (Google, OpenAI, Anthropic, atau DeepSeek).
