Overview

Film Translator Generator adalah aplikasi untuk melakukan transkripsi dan penerjemahan otomatis file video. Ini menggunakan Faster-Whisper untuk transkripsi dan berbagai model AI untuk terjemahan.

Transkripsi Otomatis

Menggunakan Faster-Whisper untuk transcribe dialog dari video dengan akurasi tinggi

Terjemahan AI

Mendukung Gemini, OpenAI, Anthropic, dan DeepSeek untuk terjemahan berkualitas tinggi

Editor Subtitle

Editor built-in untuk melakukan penyesuaian pada terjemahan

Styling Subtitle

Kustomisasi tampilan subtitle dengan berbagai opsi styling

Getting Started

Panduan langkah demi langkah untuk memulai dengan Film Translator Generator:

Persyaratan Sistem

  • Windows 10 atau lebih baru (dukungan Linux dan macOS masih dalam pengembangan)
  • Minimum 4GB RAM (8GB atau lebih direkomendasikan)
  • Untuk akselerasi GPU: NVIDIA dengan dukungan CUDA (opsional)
  • Python 3.8 atau lebih baru
  • FFmpeg terinstal dan tersedia di PATH

Instalasi

  1. Clone atau download repository dari GitHub
  2. Install dependencies dengan perintah: 
    pip install -r requirements.txt
  3. Jalankan aplikasi dengan perintah: 
    python main.py

Memulai Proyek Baru

  1. Klik "Add Video(s)" untuk menambahkan video ke antrian
  2. Pilih bahasa target dan model AI untuk terjemahan
  3. Klik "Process Selected Video" untuk memulai transkripsi dan terjemahan
  4. Setelah selesai, teks terjemahan akan ditampilkan dan dapat diedit atau disimpan

Transcription

Film Translator Generator menggunakan Faster-Whisper (implementasi optimized dari OpenAI Whisper) untuk transkripsi yang cepat dan akurat.

Model Whisper

Model Ukuran Kualitas RAM yang Dibutuhkan
tiny ~75 MB Dasar ~1 GB
base ~150 MB Baik ~1 GB
small ~500 MB Sangat Baik ~2 GB
medium ~1.5 GB Hampir Terbaik ~5 GB
large-v2 ~3 GB Terbaik ~10 GB
large-v3 ~3 GB Terbaru & Terbaik ~10 GB

Opsi Device

  • CUDA: Gunakan GPU NVIDIA untuk akselerasi transkripsi (jauh lebih cepat)
  • CPU: Jika tidak memiliki GPU, transkripsi akan berjalan pada CPU (lebih lambat)

Compute Type

Pilihan compute type mempengaruhi kecepatan dan penggunaan memori:

  • float16: Precision penuh, rekomendasi untuk GPU
  • int8_float16: Precision campuran, lebih hemat memori
  • int8: Precision terendah, paling hemat memori
  • float32: Untuk CPU

Translation

Film Translator Generator mendukung beberapa model AI untuk terjemahan:

Provider AI yang Didukung

  • Gemini AI: Model Google terbaru (Gemini Flash, Gemini Pro)
  • OpenAI: GPT 3.5, GPT-4, GPT-4o, dll
  • Anthropic: Claude 3 Opus, Sonnet, Haiku
  • DeepSeek: DeepSeek Chat

Pengaturan Terjemahan

Anda dapat mengatur beberapa parameter untuk mengoptimalkan hasil terjemahan:

  • Batch Size: Jumlah segmen yang diterjemahkan sekaligus (default: 500)
  • Temperature: Mengontrol kreativitas output (0.0-1.0)
  • Top P: Mengontrol determinisme output (0.0-1.0)
  • Top K: Filter jumlah token untuk sampling

Catatan: Setiap provider membutuhkan API key masing-masing. Lihat Konfigurasi API untuk petunjuk lebih lanjut.

Subtitle Styling

Film Translator Generator menawarkan berbagai opsi untuk styling subtitle:

Opsi Styling

  • Font: Arial, Helvetica, Times New Roman, Verdana, Tahoma, Impact
  • Warna: Putih, Kuning, Cyan, Lime, Pink, Light Gray
  • Ukuran: 12px hingga 32px
  • Posisi: Bawah, Tengah, Atas
  • Outline: Warna dan ketebalan outline
  • Background: Warna latar dan transparansi

Setelah menerapkan style, Anda dapat melihat pratinjau dan menyimpan dalam format yang didukung.

Format Output

  • SRT: Format subtitle standar yang didukung banyak pemutar video
  • VTT: Web Video Text Tracks, untuk streaming web
  • TXT: Teks biasa dengan timestamp
