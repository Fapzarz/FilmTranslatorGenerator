Masalah Umum

Berikut adalah solusi untuk beberapa masalah yang sering terjadi saat menggunakan Film Translator Generator.

Aplikasi tidak dapat diluncurkan

Periksa hal-hal berikut:

  1. Pastikan semua dependensi telah terinstal dengan benar: pip install -r requirements.txt
  2. Periksa log error pada terminal atau konsol
  3. Pastikan Python versi 3.8 atau lebih baru terinstal dan terdeteksi
  4. Coba jalankan dengan mode verbose: python main.py --verbose

Video tidak dapat diproses

Jika video tidak dapat diproses, coba langkah-langkah berikut:

  1. Pastikan format video didukung (MP4, AVI, MKV, MOV, atau WebM)
  2. Periksa jika video tidak rusak dengan membukanya di pemutar video lain
  3. Coba konversi video ke format MP4 dengan FFmpeg: ffmpeg -i input.mkv -c:v libx264 -c:a aac output.mp4
  4. Periksa apakah file video mungkin terkunci oleh aplikasi lain

Transkripsi gagal atau tidak akurat

Masalah transkripsi dapat disebabkan oleh:

  • Kualitas audio yang buruk - coba normalisasi audio terlebih dahulu
  • Model Whisper yang terlalu kecil untuk tugas - coba model yang lebih besar (seperti "large-v3")
  • Kurangnya memori GPU - coba gunakan "int8" compute type atau pindah ke CPU
  • Bahasa yang tidak terdeteksi secara otomatis - pastikan bahasa dipilih dengan benar

Anda juga dapat menggunakan diagnose tool yang tersedia dengan mengklik kanan pada video dan memilih "Diagnose Video Playback".

Terjemahan gagal atau error API

Jika terjemahan gagal, periksa hal-hal berikut:

  • Pastikan API key valid dan tidak kedaluwarsa
  • Periksa koneksi internet Anda
  • Periksa kuota API Anda (mungkin Anda telah mencapai batas)
  • Coba model terjemahan alternatif (misalnya, dari Gemini ke OpenAI)
  • Kurangi batch size untuk menghindari timeout API

Tampilan video atau preview subtitle tidak berfungsi

Masalah preview dapat disebabkan oleh:

  • Codec video yang tidak terpasang - instal codec media yang diperlukan
  • Format video yang tidak didukung - konversi ke format yang lebih umum
  • Konflik dengan driver grafis - perbarui driver grafis
  • Konflik dengan PySide6 - pastikan versi PySide6 kompatibel dengan sistem Anda

Kode Error

Film Translator Generator menyediakan kode error yang dapat membantu mengidentifikasi masalah spesifik.

Kode Error Deskripsi Solusi
ERR_TRANSCRIBE_001 Gagal memuat model Whisper Periksa koneksi internet dan ruang disk. Model Whisper cukup besar.
ERR_TRANSCRIBE_002 Kesalahan saat ekstraksi audio Pastikan FFmpeg terinstal dan video tidak rusak.
ERR_API_001 API key tidak valid Periksa dan perbarui API key di pengaturan.
ERR_API_002 Batas kuota API tercapai Tunggu hingga kuota direset atau beli kuota tambahan dari penyedia API.
ERR_API_003 Timeout koneksi API Periksa koneksi internet atau coba lagi nanti.
ERR_FILE_001 Format file tidak didukung Konversikan file ke format yang didukung (MP4, AVI, MKV, MOV, WebM).
ERR_FILE_002 Gagal membaca file Pastikan file tidak rusak dan dapat diakses.
ERR_SAVE_001 Gagal menyimpan file output Periksa izin folder dan ruang disk yang tersedia.
ERR_CUDA_001 Error CUDA saat transkripsi Perbarui driver GPU atau pilih opsi CPU sebagai device.
ERR_MEM_001 Memori tidak cukup Tutup aplikasi lain, kurangi ukuran model, atau gunakan compute type yang lebih efisien.

Diagnostik

Film Translator Generator menyediakan beberapa alat diagnostik untuk membantu memecahkan masalah.

Video Diagnostic Tool

Untuk mendiagnosis masalah dengan file video tertentu:

  1. Klik kanan pada video di daftar antrian
  2. Pilih "Diagnose Video Playback"
  3. Tinjau hasil diagnostik di log

Log Diagnostik

Log aplikasi dapat memberikan informasi penting tentang masalah:

  • Log disimpan di folder logs/ dengan format tanggal
  • Gunakan level logging verbose dengan menjalankan python main.py --debug
  • Untuk masalah API, periksa log untuk kode respons HTTP dan pesan error

Tes Koneksi API

Untuk menguji koneksi ke API tanpa memproses video penuh:

  1. Buka menu "Tools"
  2. Pilih "Test API Connection"
  3. Pilih provider API yang ingin diuji
  4. Klik "Test" untuk memverifikasi koneksi dan key

Persyaratan Sistem

Pastikan sistem Anda memenuhi persyaratan minimum berikut:

Persyaratan Minimum

  • OS: Windows 10, macOS 10.15+, atau Linux (Ubuntu 20.04+)
  • Prosesor: Quad-core CPU 2.0 GHz atau lebih baik
  • RAM: 4GB (8GB direkomendasikan)
  • Penyimpanan: 2GB ruang disk tersedia (10GB direkomendasikan untuk model)
  • Python: 3.8 atau lebih baru
  • FFmpeg: Terinstal dan tersedia di PATH

Persyaratan untuk Akselerasi GPU

  • NVIDIA GPU dengan CUDA 11.2 atau lebih baru
  • 4GB VRAM minimum (8GB+ direkomendasikan untuk model large)
  • Driver NVIDIA up-to-date

Dependensi Software

Film Translator Generator membutuhkan beberapa dependensi utama berikut:

  • FFmpeg untuk pemrosesan audio/video
  • PySide6 untuk antarmuka pengguna
  • faster-whisper untuk transkripsi
  • torch untuk framework machine learning

Jika Anda menghadapi masalah performa, coba gunakan model Whisper yang lebih kecil atau alihkan ke CPU jika GPU Anda tidak memadai.

Bantuan Komunitas

Jika Anda tidak dapat menyelesaikan masalah dengan panduan troubleshooting ini, silakan hubungi komunitas:

  • Buat issue di GitHub Issues
  • Periksa GitHub Discussions untuk solusi umum
  • Periksa CHANGELOG.md untuk masalah yang diketahui dan solusi dalam rilis terbaru

Saat Melaporkan Bug

Untuk mendapatkan bantuan yang lebih baik, sertakan informasi berikut saat melaporkan bug:

  1. Versi Film Translator Generator
  2. Sistem operasi dan versi
  3. Spesifikasi hardware (RAM, GPU jika relevan)
  4. Output log lengkap (disalin dari konsol atau dari file log)
  5. Langkah-langkah yang dapat direproduksi untuk memunculkan masalah
  6. Screenshot dari error (jika tersedia)
