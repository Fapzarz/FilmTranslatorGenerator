Advanced Subtitle Editor

Film Translator Generator menyediakan editor subtitle lanjutan untuk pengaturan yang lebih presisi dan fleksibel.

Mengakses Editor Lanjutan

Anda dapat mengakses editor subtitle lanjutan dengan:

  1. Klik kanan pada video dalam antrian
  2. Pilih "Open Advanced Subtitle Editor"

Fitur Editor Lanjutan

  • Pengeditan Timestamp: Sesuaikan waktu mulai dan selesai untuk setiap segmen subtitle
  • Split & Merge Segments: Gabungkan beberapa segmen atau bagi segmen panjang menjadi lebih kecil
  • Inline Styling: Terapkan format ke bagian spesifik dari teks
  • Kontekstual Editing: Edit dengan konteks segmen sebelumnya dan berikutnya
  • Preview Sync: Lihat subtitle pada video saat mengedit

Perubahan yang dibuat di editor lanjutan akan diaplikasikan ke dalam proyek saat Anda klik "Apply Changes".

  5. Klik "Apply" untuk menyimpan perubahan

Batch Processing

Film Translator Generator memungkinkan pemrosesan batch untuk beberapa video sekaligus.

Mengatur Batch Processing

  1. Tambahkan beberapa video ke antrian
  2. Atur semua parameter yang diinginkan (bahasa target, model, dll.)
  3. Klik menu "Processing" lalu pilih "Process All Videos"

Opsi Batch Processing

  • Sequential Processing: Proses video satu per satu (default)
  • Parallel Processing: Proses beberapa video sekaligus (perlu perangkat keras yang lebih kuat)
  • Auto Save: Simpan hasil otomatis ke file setelah pemrosesan

Parallel processing membutuhkan lebih banyak sumber daya sistem dan mungkin tidak stabil pada perangkat dengan spesifikasi rendah.

Translation Memory

Translation Memory memungkinkan Film Translator Generator menggunakan hasil terjemahan sebelumnya untuk meningkatkan konsistensi dan mengurangi biaya API.

Cara Kerja Translation Memory

Saat Anda menerjemahkan segment, sistem akan:

  1. Menyimpan pasangan teks sumber dan terjemahan
  2. Mencari kecocokan saat ada teks serupa di masa depan
  3. Menggunakan terjemahan yang tersimpan jika kecocokan melebihi threshold tertentu

Mengonfigurasi Translation Memory

Anda dapat menyesuaikan pengaturan Translation Memory di bagian Advanced Settings:

  • Threshold: Atur tingkat kecocokan minimum (default: 90%)
  • Import/Export: Impor atau ekspor TM Database
  • Clear Memory: Hapus semua entri dari Translation Memory

Custom Preprocessing

Film Translator Generator mendukung preprocessing kustom untuk hasil yang lebih baik.

Preprocessing Audio

  • Noise Reduction: Kurangi noise background untuk transkripsi yang lebih akurat
  • Audio Normalization: Sesuaikan level audio untuk konsistensi
  • Channel Selection: Pilih saluran audio spesifik untuk transkripsi

Preprocessing Text

  • Text Normalization: Standarisasi teks sebelum terjemahan
  • Custom Replacements: Atur penggantian teks kustom
  • Prompt Templates: Kustomisasi prompt untuk model terjemahan

Mengakses Preprocessing Settings

Untuk mengakses pengaturan preprocessing:

  1. Buka menu "Settings"
  2. Pilih "Advanced Settings"
  3. Navigasi ke tab "Preprocessing"

Project Management

Film Translator Generator menyediakan manajemen proyek yang komprehensif untuk mengatur dan melacak pekerjaan terjemahan Anda.

Struktur Project

Project (.ftgproj) menyimpan informasi berikut:

  • Video Queue: Daftar file video yang ditambahkan
  • Processing Status: Status setiap file dalam antrian
  • Transcription Data: Hasil transkripsi untuk setiap video
  • Translation Data: Hasil terjemahan untuk setiap video
  • Styling Settings: Pengaturan styling subtitle
  • Output Settings: Format output dan opsi lainnya

Project Backup & Recovery

Film Translator Generator melakukan backup otomatis dari proyek Anda dengan:

  • Auto Save: Simpan proyek pada interval tertentu
  • Backup Files: Buat file cadangan dengan timestamp
  • Project Recovery: Pulihkan dari backup jika terjadi crash

File proyek (.ftgproj) mengenkripsi semua API key dengan aman, sehingga dapat dibagikan tanpa risiko keamanan.
